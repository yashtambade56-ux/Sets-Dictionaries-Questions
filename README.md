# Sets-Dictionaries-Questions
This repository contains 9 structured Python problems based on Sets and Dictionaries, designed to help beginners understand how to efficiently store, manage, and manipulate data.
These exercises strengthen concepts like set operations, dictionary creation, lookup, updating, removing items, counting occurrences, and converting between different data collections.
This part represents Questions 29 to 38 from the full 50-question Python basics series.

✅Topics Covered
1. Creating sets from user input
2. Union, intersection & difference operations
3. Unique elements using sets
4. Creating dictionaries
5. Updating, deleting key–value pairs
6. Finding max values (topper problem)
7. Word frequency counting using dict
8. Converting lists to dictionary
9. Reversing dictionaries
10.Comparing values between dictionaries
11.Counting unique values
12.Looping through nested dictionary data
